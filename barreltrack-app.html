Google Sheets Connection
Enter your Google Cloud credentials to connect BarrelTrack to your barrel inventory spreadsheet. These are the same credentials used in the Sheets Config tool.
From Google Cloud Console → APIs & Services → Credentials → OAuth 2.0 Client ID
From the Google Sheet URL between /d/ and /edit
Where to find these values
Client ID: console.cloud.google.com → APIs & Services → Credentials → click your OAuth 2.0 Client ID name
Spreadsheet ID: Open your Google Sheet → look at the URL bar → copy the long string between /d/ and /edit
Sheet Tab Name: The tab name at the bottom of your Google Sheet (default: Barrels)
🔍 Stored Credential Diagnostic
Tap "Check Stored Credentials" to verify what is saved.
⚠️ If you see Error 401: invalid_client
1. Tap Check Stored Credentials above — verify the Client ID shown matches Google Cloud Console exactly
2. Go to console.cloud.google.com → APIs & Services → Credentials
3. Click your OAuth 2.0 Client ID name → copy the Client ID field
4. Tap Clear & Re-enter above, then paste it fresh
5. Confirm your email is added as a Test User on the OAuth consent screen
